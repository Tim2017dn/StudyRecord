# 指针

## 初步认识

### 正常写法

``` c
  int s1=100;
    int* p1;
    p1=&s1;

    printf("p1指向的值： %d",*p1);
```

```
p1指向的值： 100
```



### 其他写法的情况

#### 写法一

``` c
    int s1=100;
    int* p1=&s1;

    printf("p1指向的值： %d",*p1);
```

```
p1指向的值： 100
```

没问题

#### 写法二

```c 
  int s1=100;
    int* p1=s1;
//    p1=&s1;
    printf("p1的值： %d",p1);
    printf("p1指向的值： %d",*p1);
```

编译会出现问题，任由一定的返回

```
p1的值： 100
```

相当于从 int* 变为 int
