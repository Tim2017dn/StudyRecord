# 占位符

%d	十进制整数 int

%ld	长整型的 long

%u	无符号整数

%o	八进制整数

%x	十六进制整数

%s	字符串

```
%3.4s  3表示宽度 4表示精度
%-8s	-表示左对齐
```



%f	由于输入带小数 float double

%lf	long double

%p	指针

%c	字符

%a	浮点数





## 输出形式

```cc
printf("%d",x);
printf("%3d",x);		要求3位整型
printf("%*d",i,x)		i替代3位置 i位整型
```
