# git使用

## 初始化
```
git init
```

## 拷贝远程仓库
```
git clone <url>
git remote add origin <url>
```

## 添加
```
git add .
```

## commit
```
git commit -m "some information"
```

## 同步仓库
```
git pull
```

## 推到远程仓库
```
git push
git push -u origin master
```
